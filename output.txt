To enter an arithmetic, you must type it in prefix format.
Example: + A B = A + B
Use + for addition
Use - for subtraction
Use * for multiplication
Use / for division
Use ^ for exponent
Use ' for rounding up
Use _ for rounding down
Use # for rounding to nearest integer
Use % for modulo
